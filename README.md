# LingkodHub: Digital Barangay Service Platform

LingkodHub is a web-based system designed to digitalize barangay services by allowing residents to request documents, schedule appointments, and track application status online. It also provides administrators with tools to efficiently manage requests and appointments.

---

## 🚀 Features

- 📄 Online Document Request System
- 📅 Appointment Booking System
- 📊 User Dashboard (Track requests & appointments)
- 🛠️ Admin Dashboard (Manage requests & approvals)
- ⏱️ Real-time Status Updates (Pending, Approved, Rejected)
- 🔐 Role-based Access Control (Admin & Resident)

---
